# Prime Porkour

A Pygame-based game about a pig and prime numbers.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository_url>
    cd prime-porkour
    ```
2.  **Install dependencies using uv:**
    ```bash
    uv pip install -r requirements.txt
    ```

## Running the Game

```bash
uv run python main.py
```
